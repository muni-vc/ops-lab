01 — Overview
The Work at a Glance
Alongside managing EMEA content operations, a growing suite of tools, systems, and AI workflows has been independently designed and built across operations — each one solving a real problem identified through daily work.
Approach
"Every workflow gap is an opportunity. I don't flag problems and move on — I identify them, design the fix, and build it."
Every tool and system here was self-initiated — no brief, no external instruction. Each one traces back to a specific pain point found through day-to-day operations: manual file naming across hundreds of assets, repetitive task creation every week, fragmented partner file exchanges, multi-step graph generation processes that could be collapsed into one. The principle behind all of it is the same — if something is being done manually and repeatedly, it should not be.

What This Replaces
Hours of Manual Work
File renaming, folder creation, task setup, graph generation, report publishing, brief cleanup — all previously done by hand, every cycle. The time saved compounds across every project, every week, across every market.
How It Was Built
Learning by Doing
No formal coding background — everything here was built through active learning, picking up new tools and approaches as needed, and working with AI and vibe coding to turn operational ideas into working systems.
⚙️
Built entirely using free-tier tools. Every tool, dashboard, and AI workflow here runs without API charges or external subscriptions — built using Google Apps Script, Python, HTML, and the Gemini free API. Full operational value at zero running cost.
